# Triage Labels

工程技能使用五个标准 triage 角色。本文件把这些角色映射到本仓库 issue tracker 中实际使用的标签或状态字符串。

| Label in mattpocock/skills | Label in our tracker | 含义 |
| -------------------------- | -------------------- | ---- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，AFK agent 可以直接处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色时，例如“apply the AFK-ready triage label”，使用上表中 `Label in our tracker` 对应的字符串。

如果以后项目改用其他标签体系，直接编辑右侧列即可。
